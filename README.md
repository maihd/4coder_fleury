# MaiHD's 4coder Custom Player
Heavily based on Ryan Fleury's Personal 4coder Custom Layer

## Build
1. `cd path/to/4coder/custom`
2. `git clone https://github.com/dion-systems/4coder_fleury`
3. `cd 4coder_fleury && build`
